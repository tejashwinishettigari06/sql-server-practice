***CODING ORDER***

**------------**



**1. SELECT** --- filter columns

**2. DISTINCT** --- filter duplicates

**3. TOP** : Filter result rows

&#x09;col1,

&#x09;sum(col2)

**4. FROM** Table

**5. WHERE** col=10 -- filter rows before aggregations

**6. GROUP BY** : col1

**7. HAVING** sum(col2) > 30 ---filter rows after aggregations

**8. ORDER BY** col1 ASC
